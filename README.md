# wechat-jump-helper
微信跳一跳半自动鼠标操作Electron版


## 使用说明

使用命令行工具执行下面的命令

```bash
# Clone this repository
git clone https://github.com/juemeng/wechat-jump-helper
# Go into the repository
cd wechat-jump-helper
# Install dependencies
npm install
# Run the app
npm start
```

Note:如果需要生成可执行文件，执行：
```
npm run dist or yarn dist
```
